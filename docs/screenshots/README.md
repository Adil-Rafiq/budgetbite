# Screenshots

The repo-root `README.md` references the following images. Drop them in this directory using these exact filenames so the README renders properly on GitHub.

| File                    | What to capture                                                                                                                       | Notes                            |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------- |
| `landing.png`           | The marketing landing page hero (`/` while logged out).                                                                               | 1600x900 or wider works well.    |
| `dashboard.png`         | The signed-in dashboard (`/dashboard`) showing the active plan, budget summary, and meal slots for the day.                           | Use a plan with real-looking data. |
| `plan-detail.png`       | A budget plan detail page (`/plans/[id]`) including the summary card, generated suggestions, and the timeline.                        |                                  |
| `restaurants.png`       | Restaurants list (`/restaurants`) with the map, filters, and pagination.                                                              |                                  |
| `analytics.png`         | Analytics page (`/analytics`) — spending chart + meal history.                                                                        |                                  |
| `onboarding.png`        | One of the onboarding steps (the XState wizard). The location-picker step is the most visually interesting.                           |                                  |

PNG is preferred over JPG (sharper text). Keep each file under ~600 KB if possible; resize/compress with tinypng.com or `oxipng` if needed.

Until these are added, the README image references will appear as broken links in editors but will render gracefully (as alt text) on GitHub.
